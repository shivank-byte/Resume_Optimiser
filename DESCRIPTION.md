# Project Description (short form)

Use this for a portfolio card, LinkedIn project entry, or resume line.

---

**Resume Optimizer AI**
Prompt engineering project demonstrating role-based prompting, few-shot
calibration, and forced structured JSON output. A single system prompt
casts an LLM as both an ATS parser and a senior recruiter, returning an
ATS compatibility score, job-match score, missing keywords, and XYZ-formula
bullet rewrites — with explicit guardrails against hallucinated metrics.
Built with Python, the Gemini API, and Streamlit; deployed as a live web app.

**One-liner (for a resume bullet):**
Designed and deployed a structured-output prompt engineering pipeline
(role-based prompting + few-shot calibration) that scores resumes against
ATS and recruiter criteria via the Gemini API, enforcing strict JSON
contracts and anti-hallucination constraints; built with Python and
deployed on Streamlit Cloud.
